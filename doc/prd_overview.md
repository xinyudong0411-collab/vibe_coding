# 📄 PRD：基金持仓智能决策Agent（Overview）

## 1. 产品定位

一个帮助用户管理基金持仓、分析市场变化并辅助做T决策的智能Dashboard系统。

---

## 2. 核心功能模块

1. OCR截图导入持仓
2. 持仓Dashboard分析
3. 决策引擎（做T/风险判断）
4. Agent对话解释系统

---

## 3. 用户画像

- 短线 + 波段 + 做T用户
- 风格偏稳健
- 持仓5–30只基金
- 不擅长系统分析

---

## 4. 核心价值

- 降低信息处理成本
- 提供结构化决策建议
- 减少情绪化操作

---

## 5. 使用流程

上传持仓 → 自动分析 → Dashboard → 决策建议 → 用户操作

---

## 6. 非目标

- 不做自动交易
- 不保证收益
- 不做高频量化
